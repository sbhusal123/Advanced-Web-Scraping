# Splash Container Management:
- **Download Splash Image:** ``docker pull scrapinghub/splash``
- **Start Splash Container:** ``docker run -d --name splash -p 8050:8050 scrapinghub/splash``
- **Stop Splash Container:** ``docker stop splash``
